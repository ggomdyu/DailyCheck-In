# DailyCheck-In
Do the daily check-in automatically on eCommerce sites, game sites, or anywhere!

## How to use
### gmarket
```
python gmarket_daily_checkin.py your_id your_password
```

### IchikaDailyJanken
```
dotnet IchikaDailyJanken.dll your_email your_password
```

### NonoCardGame
```
dotnet IchikaDailyJanken.dll your_email your_password
```
